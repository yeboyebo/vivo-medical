=== WooCommerce Realex Payments HPP ===
Author: skyverge, woocommerce
Tags: woocommerce
Requires PHP: 5.3
Requires at least: 4.4
Tested up to: 4.9.8

Adds Realex Payments HPP to your WooCommerce website.

See http://docs.woothemes.com/document/realex-redirec-payment-gateway/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-realex-redirect' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
